Description

Tous les mois, la RATP récupère des informations à partir du graphique du site Vigicrues. Le problème, c’est le temps que cette action prend ! Cette application nommée “30joursvigicrues” affiche les informations que récupère la RATP et les formatent dans un tableau. Cela permet de récupérer directement ces informations et de les transvaser dans un fichier excel.
