Description

Les algorithmes de machine learning de détection d’objets sont de plus en plus répandus aujourd'hui et requièrent une grande quantité de données pour être entraînés. Cette application permet de labelliser les objets d’une image en ajoutant des “boundary box” autour de ces images.

Fonctionnalités

  • API RESTful avec Python et Django
  • Frontend avec React
  • Système d’authentification basé sur les tokens
  • HTML Canvas pour afficher l’image et gérer les rectangles
